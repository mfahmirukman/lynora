# Lynora 0.4 — Web + Optional Sync

**Goal:** Browser workbench (REST/GraphQL), optional account sync (last-write-wins + local backup), desktop sign-in, Linux/macOS bundle targets documented.

**Architecture:**
- `lynora-sync` — account + collection bundle push/pull client + shared types
- `apps/lynora-sync-server` — Axum self-host sync API (SQLite)
- `apps/lynora-web` — Vite TS UI; IndexedDB local store; browser fetch for REST/GraphQL; sync client via HTTP
- Desktop — sign-in + sync commands wrapping `lynora-sync`
- gRPC in browser: deferred (desktop-only note in UI)

**Conflict policy:** last-write-wins with `.bak-<timestamp>` local snapshot before overwrite.
